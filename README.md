# RecyclerView
    在开发过程中，经常会遇到各种奇葩的规则，此处将一些常用规则整理并封装

## 内容
1. **RichRecyclerView**--结合**LinearLayout**和**RecyclerView**,支持特性：**loading**,**loadFailure**,**empty**,**normal**
2. **RefreshRecyclerView**--结合**SwipeRefreshLayout**和**RichRecyclerView**,支持：**pull-refresh**
